Data files: 
MCQ_E/F/G.XPT - Diagnoses data extracted from NHANES

Matlab script: 
get_diag_data.m - Identifies the patients with coronary heart disease, stroke and heart failure.

Output files:
chd_pat.txt - Two-column ASCII file with ID and coronary heart disease status.
stroke_pat.txt - Two-column ASCII file with ID and stroke status.
hf_pat.txt - Two-column ASCII file with ID and heart failure status.
