Data files: 
BIOPRO_E/F/G.XPT - Standard Biochemistry profile (of lab data) from NHANES

Matlab script: 
egfr_calc.m - Calculates the eGFR from serum creatinine (extracted from the BIOPRO files) and the previously extracted demographic data in (in the demog_data folder).

Output file:
egfr_table.txt - Two-column ASCII file with ID and calculated eGFR respectively.
