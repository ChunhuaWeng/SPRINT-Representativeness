Data files: 
ALB_CR_E/F/G.XPT - Urine albumin and creatinine data from NHANES

Matlab script: 
get_proturia.m - Identifies patients with proteinuria based on urine albumin and creatinine extracted from the ALB_CR files.

Output file:
proturia_pat.txt - Two-column ASCII file with ID and proteinuria status.
albcr_data.txt - Two-column ASCII file with ID and Albumin-Creatinine measurement.
